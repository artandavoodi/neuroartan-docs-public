Source Calibration

Source Calibration is a private Foundation assessment that helps the model understand how a person currently interprets agency, control, regulation, flexibility, and continuity.

It is not a diagnosis, therapy, spiritual framework, public score, or identity label. It is a private model-foundation signal used to improve reflection, pacing, memory linking, and support style.

What Source Calibration measures

  • Cognitive Orientation Index
  • Dominant Orientation
  • Control Orientation
  • Agency Level
  • Regulation Style
  • Cognitive Flexibility
  • Narrative Coherence

How scoring works

Each answer is scored on a continuous scale from 0 to 10. The system calculates dimension-level scores, identifies the strongest orientation pattern, and produces a private Source Profile.

Status indicators follow a score-driven logic: 0–3.99 indicates an initial signal, 4–6.99 indicates a forming signal, and 7–10 indicates a stronger or more stable signal.

Privacy

Source Calibration results remain private to the user’s model foundation unless future sharing controls are explicitly enabled by the user and approved by governance.
