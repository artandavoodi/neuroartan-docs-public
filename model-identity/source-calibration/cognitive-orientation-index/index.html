Cognitive Orientation Index

Cognitive Orientation Index is the overall Source Calibration score. It summarizes how strongly the current answers indicate agency, regulation, flexibility, and coherent self-reflection.

How to read it

The score is shown on a 0 to 10 scale. A higher score indicates a stronger calibrated signal across the Source dimensions. A lower score indicates that the model has less stable foundation information and should use more careful pacing.

Status logic

  • 0–3.99: Initial signal.
  • 4–6.99: Forming signal.
  • 7–10: Stronger or more stable signal.

How the model uses it

The model uses this index to choose safer starting defaults for reflection depth, pacing, memory linking, and cognitive support style.
