Control Orientation

Control Orientation describes how a person currently interprets control, influence, and responsibility. It helps the model understand whether the user’s answers suggest externalized control, mixed control, or more integrated agency.

How to read it

The label summarizes the current pattern. The score behind the label is shown on a 0 to 10 scale and determines the status color.

Why it matters

The model uses Control Orientation to choose how directly it should reflect agency, responsibility, uncertainty, and possible next action.

Important boundary

Control Orientation is not a diagnosis or fixed identity. It is a private calibration signal that can change with new answers and recalibration.
