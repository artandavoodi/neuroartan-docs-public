Privacy & Safety

Data Ownership & Portability

Neuroartan’s direction is that the user’s raw mind archive remains user-owned, portable, understandable, and removable wherever technically and legally possible.

User-owned archive

Neuroartan should not trap a person’s thoughts, memories, voice records, or continuity settings inside an opaque system. Raw source material should remain readable and exportable wherever possible.

The preferred long-term direction is a local-first vault supported by optional iCloud or encrypted Neuroartan Cloud processing when the user explicitly chooses deeper model use.

Portable formats

Neuroartan should support portable formats such as Markdown, TXT, JSON, CSV, audio files, transcripts, consent records, memory graph exports, and legacy configuration exports where feasible.

Portability matters because memory, voice, and legacy data are not ordinary app settings. They may represent a person’s identity, family history, private thought structure, and continuity intent.

Access and correction

Users should be able to understand what data exists, what it is used for, where it is stored, and whether it is local, iCloud-based, Neuroartan Cloud-based, or derived for model use.

Where appropriate, Neuroartan should provide correction workflows for profile data, model metadata, source records, memory records, consent states, and legacy configuration details.

Deletion and revocation

Deletion must address more than the original upload. It should also evaluate parsed text, chunks, embeddings, memory links, model calibration state, voice transcripts, voice-derived records, and legacy settings.

Some operational or security records may need limited retention where legally required, but that boundary must be documented and not hidden from the user.

Consent-bound processing

Neuroartan should separate consent for indexing, cloud copying, parsing, embedding generation, memory graph construction, voice processing, model calibration, and legacy continuity.

A user should be able to revoke or limit deeper processing without losing basic ownership of their raw archive.

No sale of mind-data

Neuroartan’s direction is that thought records, memory records, voice data, model calibration data, and legacy settings must not be sold, rented, traded, or used for third-party behavioral advertising.

Any future external data use would require separate governance, legal review, and explicit user consent.

Current status

This page describes product and governance direction. Specific export, deletion, correction, and portability features may depend on implementation stage and legal review.
